# Stress Check

> 대학생을 위한 스트레스 자가진단 웹앱

## 프로젝트 소개

사용자가 설문에 응답하면 스트레스 수준을 계산하여 결과와 조언을 제공하는 웹 애플리케이션입니다.

## 주요 기능

* 스트레스 자가진단
* 점수 계산
* 스트레스 수준 분석
* 맞춤형 조언 제공
* 다시 검사하기

## 사용 방법

1. 모든 문항에 응답한다.
2. 결과 보기 버튼을 누른다.
3. 스트레스 수준을 확인한다.

## 사용 예시

점수: 31점

결과: 높은 스트레스

조언: 스트레스 관리가 필요한 상태입니다. 충분한 휴식과 상담을 고려해보세요.

## 한계

* 의학적 진단 도구가 아닙니다.
* 참고용 자가진단 서비스입니다.
* 실제 정신건강 상태를 정확하게 측정하지 않습니다.
